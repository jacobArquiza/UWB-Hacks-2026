# RoRadar 30-Second Talk Track

## Read This Version

> RoRadar is a parent-facing Roblox safety tool. A parent enters a Roblox username, and our app turns public profile data, friend-network signals, game associations, and outside corroboration into a readable safety snapshot. The scoring itself is deterministic, so the core risk model is explainable, and we only use AI in a narrow second-pass role to validate whether wider web articles are actually about the exact game. That gives parents something more useful than raw data: a prioritized report showing which friends or games deserve a closer look and why.

## Even Shorter Version

> RoRadar takes public Roblox signals and turns them into an explainable parent report, with deterministic scoring and AI only used to validate ambiguous web evidence.
